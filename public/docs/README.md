# PDFs do sistema

Coloque aqui os PDFs exibidos pelo botão flutuante (FAB) do site.

Cada arquivo colocado nesta pasta é servido em `/docs/<arquivo>.pdf`.

Depois de adicionar os PDFs, edite `src/components/fm-docs.js` e ajuste o array
`PDF_DOCS` com o `file` (caminho `/docs/...`) e o `label` (nome exibido) de cada um.
